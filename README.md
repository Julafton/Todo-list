Setup:

Run "pip install -r .\requirements.txt"

Create a .env that looks like this

BACKEND_FOLDER_PATH = "The backend folder path"                                                                                                                                                                                                                  
IP = "Your local ip"                                                                                                                                                                                                                  
PORT = "The port you want the app to run on"                                                                                                                                                                                                      

Create a "users" directory in the backend folder

Run server.py
